# Problem 9: Triangle Type Checker\n## Problem Statement\nGiven sides A, B, C. Output Equilateral, Isosceles, or Scalene.\n## Clean Python Solution\n```python\ndef solve():\n    a, b, c = map(int, input().split())\n    if (a+b<=c) or (a+c<=b) or (b+c<=a): return print("Invalid")\n    if a == b == c: print("Equilateral")\n    elif a == b or b == c or a == c: print("Isosceles")\n    else: print("Scalene")\n```\n
