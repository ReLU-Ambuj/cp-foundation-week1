# Problem 6: Rock Paper Scissors O(1) Branching\n## Problem Statement\nGiven two strings representing Player 1 and Player 2 choices. Output the winner.\n## Clean Python Solution\n```python\ndef solve():\n    p1, p2 = input().split()\n    if p1 == p2: return print("Draw")\n    wins = {"Rock": "Scissors", "Scissors": "Paper", "Paper": "Rock"}\n    if wins[p1] == p2: print("Player 1")\n    else: print("Player 2")\n```\n
