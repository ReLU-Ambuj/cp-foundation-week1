# Competitive Programming Foundation: Week 1

Welcome to **Week 1: Core Syntax, Variables, Loops, and Functions**. 

This repository is designed to build an unshakable, competitive-programming-ready foundation in Python. It heavily emphasizes memory models, algorithmic constraints, edge-case engineering, and execution speed over mere syntax memorization.

## 🎯 Week 1 Objective
Transitioning from writing code that *works* to writing code that is *robust, deterministic, and optimized for CP platforms*. By the end of this week, you will be able to translate mental models of math and logic into PEP8-compliant Python instantly.

---

## 📅 The 7-Day Workflow
Every day in this repository follows a strict, repeating 4-Phase workflow designed to build muscle memory:
1. **Theory Deep Dive** (Mental models, memory architecture, Big-O concepts)
2. **Micro-Exercises** (Manual implementation of atomic concepts)
3. **Platform Problems** (Combining concepts into CP-style challenges)
4. **Timed Repetition** (Forcing recall under time constraints)

### [x] [Day 1: Core Syntax + Variables + I/O](./day1/README.md)
**Focus**: Memory models, primitive types, implicit/explicit casting, and arithmetic operators.
- **Goal**: Understand what happens in RAM when a variable is assigned and learn to read stack traces immediately without panicking.

### [x] [Day 2: Advanced Conditional Logic](./day2/README.md)
**Focus**: Boolean algebra, Control Flow Graphs (CFGs), and short-circuiting.
- **Goal**: Master Guard Clauses (early returns), flatten nested `if/else` statements, and engineer strict boundary checks for adversarial inputs.



---

## ⚙️ Strict Training Rules
1. **No IDE Autocomplete**: Disable tools like GitHub Copilot or IntelliSense. You must build raw syntactic recall.
2. **Commit Frequently**: Get into the habit of committing after every problem set using imperative tone (e.g., `feat: solve edge cases for leap year algo`).
3. **Never Skip the Speed Round**: The timed repetition at the end of each day is where surface-level knowledge turns into permanent mental architecture.
