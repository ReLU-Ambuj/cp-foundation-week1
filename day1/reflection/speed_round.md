## Instructions
Re-code the following 5 concepts from memory. **NO IDE HELP. NO AUTOCOMPLETE.** Use a simple text editor or paper.

1. Implement a complete script to read `A, B` and swap them without a temp variable using arithmetic only.
2. Implement distance between two 2D points using `math.hypot()`.

5. Recreate the bitwise Truth Tables for AND, OR, XOR.
