# Speed Re-Solve Instructions

## The Drill
1. You will randomly select **20 problems** you have solved in the past 6 days (from drills, platform problems, or the marathon).
2. You must solve them entirely from memory without opening the old files.
3. **Time Cap**: Exactly 10 minutes per problem.

## The Skip Rule
If you are stuck for 5 continuous minutes without producing a workable structural plan, **ABORT**. Skip the problem. You are training the "Recognition Reflex", and if it's not firing, sinking 30 minutes into one problem ruins the training stimulus.

## Execution
Write minimal, perfectly working code first.
Only refactor if you have time remaining on the 10-minute clock.\n
