# Console Utility Suite

## Overview
A terminal-based CLI application integrating all logic from CP Foundation Week 1. This project demonstrates strict separation of concerns, where mathematical logic is decoupled from user input handling and routing.

## Modules
1. `main.py`: The entry point and interactive menu.
2. `number_utils.py`: Pure functions for digits and number theory.
3. `prime_tool.py`: Specialized module for prime logic.
4. `calculator_tool.py`: Basic mathematical operations.
5. `pattern_tool.py`: 2D loops and structural visualization.
6. `utils/`: Input validation handlers to prevent crashes on bad data.

## Execution
Run `python3 main.py` in your terminal to start the suite.
