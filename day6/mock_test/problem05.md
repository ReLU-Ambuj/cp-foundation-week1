# Problem 05: Matrix Target Hunt

## Setup
Given an N*M matrix. Determine if Target T exists. Must use early return nested loop abstraction.

## Format
- **Input**: N M T. Then matrix.
- **Output**: True / False
- **Constraints**: N, M <= 1000
- **Difficulty**: 2/5

## Example
**In:**
```text
2 2 4\n1 2\n3 4
```
**Out:**
```text
True
```

## Checklist
- Edge Cases: T outside bounds of matrix items.
- Required Complexity: O(N*M) Worst Case
