# Problem 09: Spiral Matrix Printer

## Setup
Given N. Build an N x N matrix and fill it spirally inwards with 1 to N^2.

## Format
- **Input**: Int N
- **Output**: Matrix format.
- **Constraints**: N <= 20
- **Difficulty**: 5/5

## Example
**In:**
```text
3
```
**Out:**
```text
1 2 3\n8 9 4\n7 6 5
```

## Checklist
- Edge Cases: N=1
- Required Complexity: O(N^2)
