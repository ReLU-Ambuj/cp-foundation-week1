# Problem 06: Unique Substring Dictionary

## Setup
Given String S. Find the length of the longest contiguous substring with all unique characters.

## Format
- **Input**: S
- **Output**: Integer
- **Constraints**: S <= 10^5
- **Difficulty**: 4/5

## Example
**In:**
```text
abcabcbb
```
**Out:**
```text
3 (abc)
```

## Checklist
- Edge Cases: Empty string. All same characters.
- Required Complexity: O(N) Sliding Window equivalent
