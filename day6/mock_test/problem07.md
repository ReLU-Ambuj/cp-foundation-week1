# Problem 07: Symmetrical Array Mirror

## Setup
Is array A strictly a palindrome? Do not use list slicing `[::-1]`.

## Format
- **Input**: Length N. Array A.
- **Output**: True / False
- **Constraints**: N <= 10^5
- **Difficulty**: 2/5

## Example
**In:**
```text
5\n 1 2 3 2 1
```
**Out:**
```text
True
```

## Checklist
- Edge Cases: Empty array.
- Required Complexity: O(N)
