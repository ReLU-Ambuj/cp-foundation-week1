# Problem 01: The Case-Insensitive Palindrome

## Setup
Given string S. Return `True` if it is a palindrome ignoring all punctuation, spaces, and casing. Must be a pure function.

## Format
- **Input**: String S
- **Output**: Boolean
- **Constraints**: Len(S) <= 10^5
- **Difficulty**: 1/5

## Example
**In:**
```text
A man, a plan, a canal: Panama
```
**Out:**
```text
True
```

## Checklist
- Edge Cases: Empty string is True. String with only symbols is True.
- Required Complexity: O(N)
