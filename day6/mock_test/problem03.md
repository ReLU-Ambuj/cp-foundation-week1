# Problem 03: String Rotation Parity

## Setup
Given A and B. Can A be rotated to match B? If yes, print 'True'.

## Format
- **Input**: A, B
- **Output**: Boolean
- **Constraints**: Lengths <= 10^5
- **Difficulty**: 2/5

## Example
**In:**
```text
hello llohe
```
**Out:**
```text
True
```

## Checklist
- Edge Cases: Different lengths immediately return False.
- Required Complexity: O(N) search
