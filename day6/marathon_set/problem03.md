# Problem 03: Fast Division Count
## Problem
Given N and K. Count how many numbers from 1 to N are strictly divisible by K.

## Format
- **Input**: Two integers N, K.
- **Output**: Single integer count.
- **Constraints**: 1 <= N <= 10^9, 1 <= K <= 10^4

## Sample
**In:**
```text
10 3
```
**Out:**
```text
3
```

## Checklist before coding
- **Edge Cases Considered**: K > N (returns 0).
- **Target Complexity**: O(1) Math is required.

> **Hint**: Do NOT use a loop! Think mathematically.\n
