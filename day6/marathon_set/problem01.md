# Problem 01: The Absolute Minimum
## Problem
Given 3 integers A, B, C. Print the absolute difference between the largest and smallest numbers.

## Format
- **Input**: Three space-separated integers A, B, C.
- **Output**: A single integer.
- **Constraints**: -10^4 <= A, B, C <= 10^4

## Sample
**In:**
```text
5 -2 10
```
**Out:**
```text
12
```

## Checklist before coding
- **Edge Cases Considered**: All numbers negative, all numbers identical.
- **Target Complexity**: O(1)

> **Hint**: Use min() and max() built-ins or manual `if` branches.\n
