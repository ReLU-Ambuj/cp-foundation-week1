# Problem 06: Iterative Series Sum
## Problem
Given N. Calculate sum of 1/1 + 1/2 + 1/3 ... + 1/N. Print exactly to 3 decimal places.

## Format
- **Input**: Integer N
- **Output**: Float exactly bounded.
- **Constraints**: 1 <= N <= 10^4

## Sample
**In:**
```text
2
```
**Out:**
```text
1.500
```

## Checklist before coding
- **Edge Cases Considered**: N = 1.
- **Target Complexity**: O(N)

> **Hint**: Standard loop. Python string formatting `{res:.3f}`.\n
